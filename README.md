# svg-diagrams
HTML5+SVG+CSS3+ES5 charts rendered client-side from simple DSL markup

### Planned features:

- responsive (tv, desktop, or mobile displays)
- interactive charts (hover to focus, click to drill-down)
- animated (motion communicates detail)
